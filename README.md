# Getting Started

## Prerequisites

Node version is not strictly restricted. please make sure to use following recommended versions:
- node v16.3.2
- npm v8.1.2

## Start services in to-do-list-services

1. Install dependencies:

```
npm install
```

2. Start server:

```
npm start
```

## Start web server in to-do-list-client

1. Install dependencies:

```
npm install
```

2. Start server:

```
npm start
```
